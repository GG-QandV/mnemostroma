# .agents/

Agent configuration templates for connecting Mnemostroma to different AI environments.

Each subdirectory or file corresponds to a specific agent/IDE integration:

- `claude/` — Claude Desktop / claude.ai config snippets
- `codex/` — OpenAI Codex integration
- `custom/` — Template for custom MCP-compatible agents

Configs here are **read-only references** — copy and adapt to your setup.
