> Это руководство описывает установку расширения **без Chrome Web Store** —
> напрямую из папки в режиме разработчика. Работает в Chrome, Chromium, Edge и Firefox.

## 1. Простая установка (Рекомендуется)

Если вы установили Mnemostroma через стандартный Python-пакет (`pip`/`pipx`) или с помощью установщика Windows (`mnemostroma-clients-setup.exe`), файлы расширения **уже находятся на вашем компьютере**! Вам не нужно клонировать репозиторий Git или скачивать ZIP-архивы.

При первом запуске команды `mnemostroma setup` или установщика Windows готовая к работе скомпилированная сборка расширения автоматически распаковывается в вашу домашнюю директорию:
* **Linux / macOS**: `~/.mnemostroma/extension` (или `/home/<имя>/.mnemostroma/extension`)
* **Windows**: `%USERPROFILE%\.mnemostroma\extension` (или `C:\Users\<Имя>\.mnemostroma\extension`)

Вы можете сразу перейти к [Шагу 3 — Загрузить расширение в браузер](#4-шаг-3--загрузить-расширение-в-браузер) и указать эту папку.

## 2. Продвинутая установка (для разработчиков)

Если вы хотите участвовать в разработке расширения, изменять его исходный код, собирать собственные дистрибутивы или запускать тесты Vitest:

**Вариант A — клонировать репозиторий:**
```bash
git clone https://github.com/GG-QandV/mnemostroma.git
cd mnemostroma/src/extension
```

**Вариант B — скачать ZIP:**
1. Откройте [https://github.com/GG-QandV/mnemostroma](https://github.com/GG-QandV/mnemostroma)
2. Нажмите **Code → Download ZIP**
3. Распакуйте архив на диск
4. Путь к расширению: папка `src/extension/` (в ней должен лежать файл `manifest.json`).

Перейдите к [Шагу 3 — Загрузить расширение в браузер](#4-шаг-3--загрузить-расширение-в-браузер) и укажите эту папку.

## 3. Шаг 2 — Запустить демон Mnemostroma

Расширение отправляет перехваченные диалоги на ваш локальный демон. Без него иконка расширения будет красной.

```bash
# Проверить, что демон запущен и отвечает
curl http://127.0.0.1:8766/health
# Ожидаемый ответ: {"status":"ok","mcpConfirmed":true}
```

Если демон не запущен — обратитесь к [основному руководству по установке](https://github.com/GG-QandV/mnemostroma#readme).

## 4. Шаг 3 — Загрузить расширение в браузер

### Chrome / Chromium / Edge

1. Откройте в адресной строке браузера:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Chromium: `chromium://extensions/`

2. Включите **"Режим разработчика"** (переключатель в правом верхнем углу).

3. Нажмите кнопку **"Загрузить распакованное"** / **"Load unpacked"** (в левом верхнем углу).

4. В открывшемся диалоге выбора папки укажите путь к расширению:
   - **Для простой установки**: выберите папку `extension` внутри вашей домашней директории `.mnemostroma` (`~/.mnemostroma/extension` или `C:\Users\<Имя>\.mnemostroma\extension`).
   - **Для продвинутой установки**: выберите папку `src/extension` вашего клонированного/распакованного репозитория.

5. Расширение появится в списке с иконкой Mnemostroma.

6. Закрепите иконку в тулбаре браузера: нажмите на иконку пазла (расширения) в панели инструментов → нажмите кнопку «закрепить» (булавка) напротив Mnemostroma.

---

### Firefox

1. Откройте в адресной строке: `about:debugging`

2. Нажмите на вкладку **"This Firefox"** / **"Этот Firefox"** в левом меню.

3. Нажмите кнопку **"Load Temporary Add-on…"** / **"Загрузить временное дополнение…"**

4. Выберите файл **`manifest.json`** внутри папки расширения (либо из `~/.mnemostroma/extension/`, либо из `src/extension/`).

5. Расширение успешно загрузится и появится в списке.

> **Важно для Firefox:** временное расширение удаляется при каждом перезапуске браузера.
> Для постоянной установки без маркета используйте [Firefox Developer Edition](https://www.mozilla.org/firefox/developer/) или [Firefox ESR с подписанным XPI пакетом](https://extensionworkshop.com/documentation/publish/signing-and-distribution-overview/).

## 5. Шаг 4 — Проверить установку

1. **Иконка на панели** — нажмите на иконку Mnemostroma в тулбаре браузера.
2. **Статус подключения к демону:**
   - 🟢 зелёный — демон успешно запущен и работает.
   - 🟡 жёлтый — демон доступен, но глобальный захват диалогов приостановлен.
   - 🔴 красный — демон недоступен (убедитесь, что служба запущена).
3. **Проверить захват в реальном времени:**
   - Откройте любой поддерживаемый чат (например, [claude.ai](https://claude.ai) или [chatgpt.com](https://chatgpt.com)).
   - Начните диалог или отправьте сообщение.
   - После ответа ИИ иконка должна подтвердить успешный захват зеленой индикацией.

## 6. Обновление расширения

При обновлении Mnemostroma или изменении кода расширения:

**Chrome / Edge:**
1. Откройте страницу `chrome://extensions/`
2. Найдите блок Mnemostroma и нажмите на кнопку 🔄 (обновить) в правом нижнем углу карточки.
   *(Или нажмите кнопку «Обновить» в верхней панели страницы, чтобы перезагрузить все распакованные расширения сразу).*

**Firefox:** 
Перезагрузите временное расширение на странице `about:debugging` с помощью кнопки **"Reload"** / **"Перезапустить"**.

## 7. Устранение проблем

### Иконка расширения горит красным
Локальный демон не запущен или порты заняты другим приложением.
Выполните проверку в терминале:
```bash
curl http://127.0.0.1:8766/health
```
Если возвращается ошибка подключения — запустите демон командой `mnemostroma on`.

### Предупреждение Chrome "Расширение не из магазина"
Это стандартное предупреждение безопасности Google Chrome для любых расширений, установленных локально. Оно появляется при запуске браузера, не мешает работе и его можно просто закрыть.

### Ошибка "Could not load manifest"
Вы выбрали не ту папку при импорте. Убедитесь, что выбираете именно папку `extension` (для простого пути) или `src/extension` (для продвинутого), внутри которой непосредственно лежит файл `manifest.json` (а не наддиректории).
